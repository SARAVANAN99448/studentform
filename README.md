# Student Form Project

## Description

This project is a simple student form created using HTML, CSS, and JavaScript. The form collects basic information about students and can be used as a template for various educational or data collection purposes.

## Features

- Collects student information including name, age, gender, and email.
- Validates form inputs to ensure data integrity.
- Responsive design using CSS for a seamless user experience across devices.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/student-form.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd student-form
   ```

3. **Open the `index.html` file in your browser to view the form:**
   ```bash
   open index.html
   ```

## Project Structure

```
student-form/
├── index.html
├── styles.css
├── script.js
└── README.md
```

- `index.html`: Contains the HTML structure of the student form.
- `styles.css`: Contains the CSS styles for the form.
- `script.js`: Contains the JavaScript code for form validation and interactivity.
- `README.md`: Project documentation.
